xtls
===============================

xtls is a toolkit for `xlzd <http://xlzd.me>`_


Useage
-------

See the doc at `Documentation <http://xtls.xlzd.me>`_.


Install
-------

**Install:**

.. code-block:: console

    $ pip install xtls


**Upgrade**

.. code-block:: console

    $ pip install xtls --upgrade
